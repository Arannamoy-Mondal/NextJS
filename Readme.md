# Installation

```bash
npx create-next-app@latest ./ 
```

# Run command 

```bash
npm run dev
```


# Create nested route

```bash
mkdir src/app/route
cd src/app/route
touch page.tsx
```

localhost:3000/route


# Create dynamic route

```bash
mkdir src/app/route/[]
cd src/app/route/[]
touch page.tsx
```

localhost:3000/route
